About

Smart Parking System is designed to help drivers easily find available parking slots. The system uses digital monitoring and smart management to reduce traffic congestion and save time for drivers.

Services

✔ Real-time parking slot availability
✔ User and Admin login system
✔ QR code based vehicle entry
✔ Smart parking dashboard monitoring

Security

The system ensures security using OTP verification, admin authentication, and controlled access to the parking dashboard.

Help

If you face any issue while booking a parking slot or logging into the system, please contact the administrator or technical support team.
